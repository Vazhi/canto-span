# Research records

This directory preserves research evidence, provenance, collision audits, and
historical decision records. It is not an operational instruction directory.
Current doctrine, grammar status, workflow, and acceptance rules live under
[`../current/`](../current/) and in the root state documents.

Start with
[`CURRENT-RESEARCH-PROVENANCE.md`](CURRENT-RESEARCH-PROVENANCE.md), the
canonical package index for completed UC-RQ and repository-visible PRQ2 work.
Each listed ID has one main record, one source-verification table, and one
collision-audit table. Shared evidence spanning multiple IDs remains separate
and is linked from the overview.

Research records do not independently change parser behavior, construction
status, validation rules, or release state. When a historical record conflicts
with current doctrine or state, follow [`../current/`](../current/).
