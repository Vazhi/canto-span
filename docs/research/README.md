# Research records

This directory contains evidence, provenance, source maps, contradiction records, tests, and historical work-package snapshots.

It is **not an operational instruction directory**. Current doctrine, state, subsystem design, workflow, and acceptance rules exist under `../current/` and the root state pointers.

Interpretation rules:

- Evidence propositions and historical results remain usable only within their recorded scope.
- Historical words such as `accepted`, `supported_productive`, `promotion ready`, `evidence closed`, or `open the held-out set next` do not describe current status unless `../current/PROJECT-STATE.md` adopts them.
- Historical source and corpus counts are leads until manually reverified.
- Historical render, regression, and held-out passes are implementation evidence only.
- Historical recommendations to add motivating-example lexicon entries do not override the current compositional NP-subsystem design.
- Historical requests for new second-speaker work are suspended by the current freeze.
