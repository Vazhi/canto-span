# SV-RQ-001–SV-RQ-007 — second research round

## Scope

This round follows the first source pass with targeted searches for exact surface patterns, semantic-resource classes, overt resource topics, perfective scope, and purposive `嚟`.

The search was deliberately split into:

1. direct Cantonese scholarly analysis;
2. reference-grammar and lexicographic description;
3. primary or near-primary public Cantonese attestations;
4. pedagogical or community examples;
5. comparative evidence used only to generate diagnostics.

Search hits are attestations, not productivity evidence. Search failure is not evidence of ungrammaticality.

## Round-two headline

The largest change from round one is that the following forms now have exact independent attestations:

- resource-initial actual-use `時間用咗嚟 VP`;
- active actual-use `用咗 NP 嚟 VP`;
- overt resource topic plus agent `Resource NP + Agent + 用嚟 VP`;
- copular dedicated-function `NP 係專門用嚟 VP 嘅`.

The evidence continues to favor compositional analyses over new construction identities.

## SV-RQ-001 — resource-class licensing in `NP 用嚟 VP`

### Exact class attestations

| Resource class | Example | Source type | What it supports |
|---|---|---|---|
| funds | `啲錢用嚟交租㗎` | curated Cantonese lexicography | money may be construed as an allocated resource with a function |
| bounded time | `好多時間用嚟做嘢` | journalistic quoted speech | time intervals may be allocated to an activity |
| bounded time, actual use | `有少少時間用咗嚟等` | journalistic quoted speech | time may occur in the perfective actual-use profile |
| room/location | `間房係專門用嚟開會嘅` | dictionary example | a bounded place can receive a conventional or dedicated function |
| room/location | `會議室，用嚟開會或者舉辦活動都得` | public commercial prose | place/resource function with coordinated purposes |
| method | `呢啲方法用嚟寫記敍文` | public forum prose | a method can be presented as a resource for producing a text |
| instrument | `呢把我用嚟分線` | public forum prose | an instrument can be a left-edge resource associated with an overt user |
| opportunity | no exact independent attestation located in this round | — | remains unresolved |

### Refined semantic hypothesis

The initial NP is not restricted to a physical instrument. The attested classes share an **allocatable-resource** interpretation:

- an entity, amount, interval, place, or procedure is available or assigned;
- it is linked to a conventional, intended, or contextually assigned function;
- the VP describes that function or activity;
- an animate person is not normally interpreted as the resource without coercion, quotation, institutional-role interpretation, or another explicit relation.

This is stronger than “any nonhuman NP” and narrower than unrestricted topicalization.

### Provisional diagnostics

A candidate initial NP should be tested for:

1. **allocation:** can the referent be assigned, reserved, selected, or made available?
2. **functional relation:** can the paraphrase “the function/use of X is to VP” preserve the intended reading?
3. **agent distinction:** can a separate user be introduced without changing the resource role?
4. **conventionality versus episode:** is the clause about normal/intended function or a particular completed use?
5. **coercion:** does an animate or abstract NP require a special institutional, metonymic, or ironic interpretation?
6. **lexical alternative:** would `用 NP 嚟 VP`, `攞 NP 嚟 VP`, `留 NP 嚟 VP`, or `預留 NP 嚟 VP` be more natural?

### Current decision

`SV-RQ-001` remains a source-linked boundary question. The semantic core is **allocatable resource**, not “instrument only,” “nonhuman subject,” or unrestricted topic.

No exact support was found for treating opportunities as parallel to money, time, rooms, methods, and instruments. That class requires separate testing.

## SV-RQ-002 — overt-user resource fronting

### New exact attestation

A public Cantonese forum contains:

```text
呢把我用嚟分線
```

The surface order is exactly:

```text
Resource NP + Agent + 用嚟 + VP
```

This independently confirms that the survey’s overt-user order is not an artifact invented by the instrument.

### Structural interpretation

The attestation establishes the surface profile but does not distinguish:

- hanging topic;
- base-generated topic linked to a null object;
- moved instrument/resource;
- resumed or operator-bound resource;
- a special constructional subject.

Independent Cantonese reference grammar recognizes hanging-topic and double-subject structures. The conservative hypothesis is therefore ordinary topic-prominent syntax plus a use-function predicate.

### Diagnostic consequences

A follow-up should compare:

1. `呢把刀用嚟切麵包。`
2. `呢把刀我用嚟切麵包。`
3. `我用呢把刀嚟切麵包。`
4. `我攞呢把刀嚟切麵包。`
5. `呢把刀，我會用佢嚟切麵包。`

Required tasks:

- naturalness;
- “who uses what?” interpretation;
- contrastive or resumptive reading;
- acceptability with and without pause/prosodic boundary;
- resource classes beyond instruments.

### Current decision

The overt-user surface pattern is attested. Dedicated permanent identity is not established. Continue to test it first as topic syntax composed with a use-function clause.

## SV-RQ-003 — copular `係用嚟 VP (嘅)`

### New exact attestation

CantoDict supplies:

```text
間房係專門用嚟開會嘅。
```

This supports a dedicated-function reading containing:

- copular `係`;
- focus/specialization adverb `專門`;
- `用嚟 VP`;
- final `嘅`.

### What remains unresolved

The source does not establish whether final `嘅` is:

- required by `係…嘅` focus structure;
- preferred because of `專門`;
- optional in ordinary predication;
- omitted in neutral spoken contexts;
- a sentence-final nominalizer, assertion marker, or focus-domain closure in this pattern.

### Current decision

The survey repairs adding `嘅` align with an independently attested dedicated-function pattern, but they do not show that bare `係用嚟 VP` is ungrammatical.

Second-round classification: **compositional audit with a direct attested `係專門用嚟…嘅` subtype**.

## SV-RQ-004 — actual-use perfective `用咗嚟`

### Exact attestations located

#### Resource-initial time allocation

A Hong Kong entertainment article quotes a speaker saying:

```text
變咗有少少時間用咗嚟等
```

This is an exact resource-initial sequence:

```text
Time NP + 用咗嚟 + VP
```

It strongly supports an episodic actual-use reading: a portion of available time was in fact spent waiting.

#### Omitted-object/editorial use

A Words.hk editorial article contains:

```text
用咗嚟寫 di1
```

The exact `用咗嚟 VP` sequence is attested, though its object is discourse-recovered.

#### Active agent plus overt resource

A public pedagogical story contains:

```text
細妹用咗呢個袋仔嚟俾公仔放嘢。
```

This is not resource-initial. It is:

```text
Agent + 用咗 + Resource NP + 嚟 + VP
```

A public learner-oriented account also records:

```text
用咗英文嚟到同佢講嘢
```

This is lower-grade attestation but points in the same direction.

### Structural result

The data favor ordinary composition:

```text
perfective [用 Resource] + purposive 嚟 VP
```

or, when the resource is discourse-linked/fronted:

```text
Resource topic + perfective 用 + purposive 嚟 VP
```

The round does not justify treating `用咗嚟` as a fused aspect-purpose marker.

### Function contrast

| Form | Most plausible default |
|---|---|
| `NP 用嚟 VP` | conventional, intended, assigned, or generic function |
| `NP 用咗嚟 VP` | actual completed allocation/use of the resource |
| `Agent 用咗 NP 嚟 VP` | completed use or deployment of an overt resource for a purpose |
| `Agent 用 NP 嚟 VP` | neutral/generic instrumental-purpose relation |

### Current decision

`SV-RQ-004` advances from merely structurally plausible to **surface-attested and source-linked provisional**.

Still missing:

- controlled corpus frequency;
- variety distribution;
- constraints on resource classes;
- interaction with passive or omitted-agent readings;
- whether `嚟` can be omitted without meaning change;
- whether final particles affect acceptability.

## SV-RQ-005 — `用嚟 V咗O`

### Targeted-search result

Targeted exact-string searches for common continuations such as:

- `用嚟買咗`;
- `用嚟做咗`;
- `用嚟寫咗`;
- `用嚟切咗`;

did not yield a reliable independent Cantonese attestation in this round.

This is a search result, not negative grammatical evidence. Web indexing is incomplete, written Cantonese orthography varies, and the pattern may be context-sensitive.

### Scope hypothesis

The main semantic tension remains:

- `用嚟 VP` conventionally states a function or purpose;
- `V咗O` presents a bounded realized event.

A perfective complement may therefore require a context where the realized event is itself being identified as the purpose or actual utilization of the resource.

Potential licensing contexts:

1. retrospective identification: “What was this money used for?”
2. contrastive actual use versus intended use;
3. result reporting;
4. correction of an earlier expectation;
5. a larger `係…嘅` or explanatory structure.

### Current decision

`SV-RQ-005` remains **boundary unresolved**. The absence of easy web hits, combined with lower survey acceptance, justifies a controlled context experiment—not a ban.

The next survey must compare identical lexical frames under:

- generic-function context;
- completed-use question;
- contrastive intended-versus-actual context;
- no context.

## SV-RQ-006 — `Agent V咗 NP 嚟 VP`

### Additional exact attestations

Independent public examples include:

```text
我買咗啲雞蛋嚟整蛋撻。
```

and:

```text
我用咗好長時間嚟參悟赤霄劍法。
```

The latter is translated/game material and should not be treated as native corpus evidence, but it illustrates the same first-event-plus-purpose template.

The stronger direct descriptive sources continue to support purposive `嚟` and purpose serial constructions.

### Refined VP1 classes

Round-two attestations motivate at least these classes:

- acquisition: `買`, `攞`, `借`;
- allocation/reservation: `留`, `預留`, `用`;
- preparation/creation: potentially `整`, `準備`;
- opening/manipulation: `開`;
- motion/transport: only when the purpose relation is distinct from literal direction.

The survey’s weak `空咗時間` item remains best treated as lexical-selection failure. `預留咗時間` or `用咗時間` directly encodes allocation.

### Current decision

The purpose-serial core remains directly supported. The next boundary is lexical and event-structural, not whether perfective VP1 can precede purposive `嚟`.

## SV-RQ-007 — context-recovered `用嚟 VP` fragment

### Second-round result

No direct Cantonese construction-level source was found that decides whether bare:

```text
用嚟 VP
```

is:

- an answer fragment;
- a null-resource clause;
- topic-bound argument omission;
- ellipsis of a larger use-function clause.

Cantonese independently supports omitted pronouns and topic-prominent discourse recovery, so a full-clause analysis is possible. Fragment-answer theory remains a competing option.

### Required diagnostics

Use paired contexts:

1. **question-answer**
   - `呢樣嘢有咩用？`
   - `用嚟打蛋。`
2. **visible referent**
   - speaker points to a whisk;
   - `用嚟打蛋。`
3. **no salient referent**
   - isolated sentence;
4. **contrast**
   - `唔係用嚟打蛋，係用嚟拌麵糊。`
5. **overt resource**
   - `呢個用嚟打蛋。`

Collect:

- naturalness;
- antecedent identification;
- whether the listener supplies an omitted resource;
- whether a question is required;
- prosodic boundary and intonation.

### Current decision

`SV-RQ-007` remains source-linked provisional. No derivation is preferred after round two.

## Cross-question boundary decisions

### Established to the level of attestation

1. allocatable nonphysical resources such as money and time occur in `NP 用嚟 VP`;
2. rooms/locations can receive dedicated-function readings;
3. methods can occur as functional resources;
4. `Resource NP + Agent + 用嚟 VP` is independently attested;
5. `Time NP + 用咗嚟 VP` is independently attested;
6. `Agent + 用咗 + NP + 嚟 + VP` is independently attested;
7. copular dedicated-function `係專門用嚟…嘅` is independently attested.

### Not established

1. unrestricted productivity for all abstract nouns;
2. opportunity NPs as a stable resource class;
3. a fused `用咗嚟` construction;
4. unrestricted `用嚟 V咗O`;
5. one derivation for all resource-initial clauses;
6. one derivation for context-recovered fragments;
7. regional uniformity.

### Implementation consequence

None. These findings do not authorize runtime changes, new permanent identities, promotion, fixture edits, or release changes.
