# SV-RQ-008 follow-up design

## Neutral goal

Measure how native Cantonese speakers rate four aspectual forms across matched lexical frames and discourse contexts, without identifying a preferred form.

## Core four-form contrast

For every lexical set:

- **A:** `V完O`
- **B:** `V完O喇`
- **C:** `V咗O`
- **D:** `V完咗O`

All four conditions must keep the subject, verb, object, orthography, and context constant except for the target morphology or final particle.

## Proposed lexical classes

Use at least two lexical sets per class:

| Class | Candidate frames | Reason |
|---|---|---|
| perception/information consumption | `睇嗰本書`, `聽嗰段錄音` | compares incremental information processing |
| creation/production | `寫份報告`, `畫幅圖` | survey suggests comparatively high acceptance |
| consumption | `食個三文治`, `飲碗湯` | survey suggests lower acceptance for one consumption item |
| caused motion/transfer | `搬三個箱`, `送嗰份文件` | tests whether path/transfer structure suppresses layered aspect |
| conventional task completion | `做份功課`, `洗嗰批衫` | salient externally defined endpoint |

Avoid mixing bare mass objects with definite or quantified objects inside one lexical set.

## Context manipulation

Each lexical frame should occur in one of two counterbalanced contexts:

1. **current-result context** — someone asks whether the task is now complete;
2. **elapsed-time narrative context** — the event was completed earlier and a later state is being discussed.

A later wave may add a contrastive context such as “did it but did not finish,” but it should not be mixed into the first replication.

## Required response components

1. five-point naturalness rating;
2. optional reason categories:
   - grammar or word order;
   - lexical wording;
   - needs context;
   - interpretation differs;
   - semantic/pragmatic issue;
3. optional correction;
4. forced interpretation for `V完咗O`:
   - the event occurred and reached completion;
   - the event occurred, with completion foregrounded;
   - the form feels redundant but is understandable;
   - another interpretation;
   - cannot judge.

The options above are analytical placeholders and require blinded Cantonese wording before instrument construction.

## Counterbalancing

- Four lists.
- Each respondent sees only one form from a lexical set.
- Each form appears equally often across lexical sets and lists.
- No more than two aspect-critical items consecutively.
- Add unrelated natural, context-dependent, and degraded-but-interpretable fillers.
- Target 24–32 rated items and 8–12 minutes.

## Sampling requirement

The current survey cannot support regional inference. Before treating variety as a predictor, target minimum analyzable cells for:

- Hong Kong Cantonese;
- Guangzhou Cantonese;
- Macau Cantonese.

Record broad childhood variety and current-use variety separately if possible.

## Analysis plan

Primary outcome:

- ordinal naturalness rating.

Predictors:

- form;
- lexical class;
- context;
- object type;
- variety.

Random effects when sample size permits:

- respondent;
- lexical item.

Pre-registered comparisons:

1. D versus A;
2. D versus B;
3. D versus C;
4. form × lexical class;
5. form × context;
6. form × variety.

Do not collapse ratings into natural/unnatural for the primary model.

## Instrument safeguards

- Generated items are probes, not evidence.
- Do not expose runtime labels or expected outcomes.
- Do not use semantically anomalous controls.
- Do not infer parser changes from response proportions.
- Freeze and hash the instrument before collection.
