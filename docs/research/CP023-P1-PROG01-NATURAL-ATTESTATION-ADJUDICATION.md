# CP023-P1-PROG01 natural-attestation adjudication

Status: **occurrence threshold provisionally met; productive acceptance not established**

## Counting rule

An attestation proves that a form occurred in a bounded source context. It does not by itself prove unrestricted productivity, lexical compatibility, population frequency, naturalness for every speaker, or the parser's constituent analysis. Independence is counted by source event or corpus unit, not by repeated tokens in one document.
