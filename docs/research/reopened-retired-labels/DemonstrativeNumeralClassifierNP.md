# DemonstrativeNumeralClassifierNP

**State:** `research_pending`; no runtime authorization.  
**Historical input:** retired `DemonstrativeQuantifiedClassifierNP`.

## Narrow question

What evidence and boundaries license overt `DEMONSTRATIVE + NUMERAL + CLASSIFIER + NOUN` phrases in Cantonese?

## Why reopened

The old label had no implementation, but the nominal shape is independently legitimate. The current NP subsystem implements `DEM + CL + N` and explicitly leaves `DEM + NUM + CL + N` outside its first slice.

## Existing evidence to reverify

- `SRC-BOND-SIO-2024-CLASSIFIERS`
- `SRC-LAM-LAU-LEE-2024-SEGMENTATION`
- `SRC-XIA-2025-CLASSIFIERS`

## Required boundaries

Determine demonstrative inventory, numeral behavior, classifier compatibility, definiteness, optionality, and contrast with numeral-free demonstrative NPs. Do not infer headless forms or omitted classifiers from slot shape alone.

## Implementation safeguard

Integrate only through the shared NP subsystem after source-linked tests exist. Do not restore the dormant label as an isolated sentence template.
