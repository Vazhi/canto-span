# CP031 — v0.5.187 native-panel model migration

## Decision

The project no longer uses a fixed Speaker A / Speaker B evidence model. All
qualified respondents, including the user's wife, use the same survey
instrument and inclusion criteria and receive no special designation or
weight.

The evidence unit is now a usable adjudicated judgment per critical item.
Canonical thresholds are:

- 5–10 per item for instrument piloting;
- 10 per critical item for `provisional`;
- 30 per critical item from a locked clean instrument for
  `supported_productive`.

## Status consequence

`PostverbalZoPerfectiveVP` moves from `provisional` to `research_pending`.
Its four checked sources, executable boundaries, implementation review, one
historical private response, and ten historical public responses are preserved.
The mixed legacy instruments provide zero clean role-neutral critical-item
coverage under panel-v2, so the construction cannot retain provisional status.

`ResourceUseLaiFunctionRelation` remains `research_pending`. Its committed
23-response pilot remains useful but limited. The user reported 50 live
responses; later rows are not canonical evidence until a new snapshot is
exported, screened, and adjudicated.
