# Grammar working set

- `active/` contains the small set currently receiving substantive research or reconciliation work.
- `archived/` contains current construction records parked outside the working set.

Workflow archiving does **not** retire a runtime label, erase evidence, or change linguistic status. The canonical current construction-record set is the union of both directories. See [[CONSTRUCTION-NOTES]] and [[GRAMMAR-INDEX]].
