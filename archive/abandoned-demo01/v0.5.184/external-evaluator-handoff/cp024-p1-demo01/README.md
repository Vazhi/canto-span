# Independent custody handoff: CP024-P1-DEMO01

This folder is deliberately separate from the recovery packet's evidence and implementation materials. Its purpose is to create genuinely prospective cases that the implementing process has not seen.

Give the evaluator only `DEMO01-EVALUATOR-SPEC.md` and `seal-template.json`. The evaluator should return a sealed ZIP and checksum, not the unsealed cases. Store the sealed archive outside the Canto Span recovery tree until the candidate runtime and visible packet are frozen.
